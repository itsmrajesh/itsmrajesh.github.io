# M Rajesh — Portfolio

Personal portfolio website built with React, Vite, and Tailwind CSS.

## Tech Stack

- React 18
- Vite 6
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployed to GitHub Pages via GitHub Actions on push to `master`.

Live at: https://itsmrajesh.github.io/
